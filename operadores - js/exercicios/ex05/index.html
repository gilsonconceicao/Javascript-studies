Operadores Aritméticos

Os operadores aritméticos são aqueles que conhecemos na escola, muito simples.

Aritméticos
  • + - mais
  • - menos
  • / - divisão
  • % - resto`da divisão
  • * - multiplicação
  • ** - potência 2

Operadores Relacionais

Os operadores aritméticos são aqueles que cujo o resultado seré: verdadeiro ou falso.

Relacionais
  • > - maior
  • < - menor
  • >= - maior igual
  • <= - menor igual
  • == - igualdade
  • != - diferença

Opereadores Lógicos

Operadores lógicos existem três valores, Negação, Conjunção e Desjunção.

Lógicos
  • ! - Negação / apenas uma opção será considerada.
  • && - Conjunção / apenas valores verdadeiros.
  • || - Desjunção / basta ter apenas um verdadeiro e já será considerado.

Ternário

Operadores Ternário funciona em 3 fases: teste ? true : false
ou seja, teste? isso é verdadeiro ou falso.

Ternário
  • ? - É literalmente um dúvida exemplo: idade >= 18 ?
  • : - Os dois pontos são apenas um divisor entre verdadeiro ou falso. (verdadeiro vem primerio)
