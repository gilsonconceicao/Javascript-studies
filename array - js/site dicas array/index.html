DICAS ARRAY

O que é array?

São variáveis composta onde pode utilizar mais de um valor em uma variável. Podendo chamar através de indíces.

Como ultilizar?

Para acessar um item de um array é muito simples, basta utilizarmos o índice da posição do item. console.

Regras dos arrays:

Em geral, array é bem simples, más tem suas regras. Como, para declarar use [], e para separar os valores de cada elemento use vírgulas.
